Setup & Usage¶

Option 1: Google Colab (Recommended)¶

Click any notebook link above and open in Google Colab - no setup required!

Option 2: Local Jupyter¶

# Install dependencies
pip install jupyter numpy scipy matplotlib

# Start Jupyter
cd ~/mathai-website/docs/notebooks
jupyter notebook

Option 3: VS Code¶

Install Python and Jupyter extensions, then open .ipynb files directly.

Notebook Structure¶

Each notebook includes:

  1. Learning Objectives - What you'll learn
  2. Theory Review - Key concepts from lectures
  3. Implementation - Step-by-step code
  4. Exercises - Practice problems
  5. Solutions - Detailed explanations
  6. Extensions - Challenge problems

Best Practices¶

Before Starting: - Review corresponding lecture material - Understand the mathematical theory first - Have NumPy documentation ready

While Working: - Run cells in order - Experiment with parameters - Add your own test cases - Write notes in markdown cells

After Completing: - Compare with library implementations (SciPy, scikit-learn) - Verify numerical stability - Test edge cases - Extend with your own ideas

Getting Help¶

Common Issues: - Import errors → Install missing packages: pip install <package> - Kernel crashes → Restart kernel and clear output - Results don't match → Check array shapes and data types

Resources: - NumPy Documentation - Jupyter Documentation - Course office hours and forum
