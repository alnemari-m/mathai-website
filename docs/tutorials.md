## Tutorial Format

Each tutorial includes:

1. **Motivation** - Why this topic matters for AI/ML
2. **Mathematical Theory** - Rigorous treatment with proofs
3. **Worked Examples** - Step-by-step solutions
4. **Exercises** - Practice problems with solutions
5. **Applications** - Connections to real ML algorithms
6. **Further Reading** - References for deeper study

## How to Use These Tutorials
